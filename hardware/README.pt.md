# Hardware

Este repositório suporta múltiplas placas de desenvolvimento FPGA e componentes eletrônicos para construção de projetos. Embora nosso objetivo seja fornecer exemplos para várias plataformas, os projetos base iniciais são construídos usando o **Tang Nano 9K**.

## Placas Suportadas

### Suportadas Atualmente

| Placa | Chip | LUTs | BRAM | Características Principais |
|-------|------|------|------|---------------------------|
| [Tang Nano 9K](tang-nano-9k.md) | Gowin GW1NR-9C | 8.640 | 26,5 KB | HDMI, 2x PMOD, LEDs RGB, 4MB Flash |

### Em Breve

- Tang Nano 20K
- Alchitry Au
- ICEBreaker
- ULX3S
- Outras plataformas (sugestões são bem-vindas!)

## Primeiros Passos

Cada placa possui sua própria página de documentação com:
- Diagramas de pinout
- Guias de conexão
- Especificações de clock e I/O
- Arquivos de constraints de exemplo

Selecione sua placa da lista acima para começar.
