# Hardware

This repository supports multiple FPGA development boards and electronic components for building projects. While we aim to provide examples for various platforms, the initial base projects are built using the **Tang Nano 9K**.

## Supported Boards

### Currently Supported

| Board | Chip | LUTs | BRAM | Key Features |
|-------|------|------|------|--------------|
| [Tang Nano 9K](tang-nano-9k.md) | Gowin GW1NR-9C | 8,640 | 26.5 KB | HDMI, 2x PMOD, RGB LEDs, 4MB Flash |

### Coming Soon

- Tang Nano 20K
- Alchitry Au
- ICEBreaker
- ULX3S
- Other platforms (suggestions welcome!)

## Getting Started

Each board has its own documentation page with:
- Pinout diagrams
- Connection guides
- Clock and I/O specifications
- Example constraints files

Select your board from the list above to get started.
